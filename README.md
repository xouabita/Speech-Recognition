Speech Recognition
==================

### Dependencies

- libsndfile
- gnuplot

#### Installation OS X

```bash
brew install libsndfile
brew install gnuplot
```

#### Installation ubuntu
```bash
sudo apt-get install libsndfile1 libsndfile1-dev gnuplot clang
```

### Usage

```bash
make visualize
./visualize # output: waveform.png
```
